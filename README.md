# Project-LORA-BASED ENVIRONMENTAL SURVEILLANCE SYSTEM FOR DURIAN ORCHARDS
A LoRa-based environmental monitoring system for durian orchards.  
Using ChirpStack, ESP32, and SX1276 for data transmission.
